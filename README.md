##Установка проекта
1. Склонируйте к себе git репозиторий с помощью команды 

'git clone https://github.com/Aheguy4ik/gittusk1'

2. Запустите любой файл проекта с помощью команды 

'python3 <название_файла>'

##Использование проекта
